# sqOS

sqOS is JavaScript-based virtual-OS project created for fun.

Contributors:
- [Moka](https://github.com/DawnStar907)

## Features (At the moment):

- Calculator 
- Dock 
- Various seal wallpapers
- Movable windows 
- Fullscreen windows
